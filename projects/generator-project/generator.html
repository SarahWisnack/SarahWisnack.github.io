GENERATOR PROJECT

When the user refreshes the page they will get a new letter each time. The algorithm will cycle through the entire english alphabet. When the page appears it will be blank with a slider and as the user moves the slider, rings will form the letter. If the user clicks and drags the slider to the right, the letter will become more complete. When the user clicks and drags the slider to the left, there will be less rings resulting in there being no letter formed.
